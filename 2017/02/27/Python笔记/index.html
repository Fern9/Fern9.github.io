下划线

__foo__:一种约定,Python内部的名字,用来区别其他用户自定义的命名,以防冲突.

_foo:一种约定,用来指定变量私有.程序员用来指定私有变量的一种方式.

__foo:这个有真正的意义:解析器用_classname__foo来代替这个名字,以区别和其他类相同的命名.

装饰器

new-style and old-style

Python3中全部为新式类

区别

  • 新式类__class__和type都返回 __main__.ClassName
  • 旧式类__class__ 返回 __main__.ClassName，type返回<type 'instance'>
  • 新式类继承时广度优先，旧式类继承时深度优先

解决方法

  • 声明__metaclass__ = type
  • 继承新式类

__new__ and __init__

  • __new__为静态方法，返回实例，用来创建实例，实例创建完之后再执行__init__
  • __init__为实例方法，初始化实例属性值
